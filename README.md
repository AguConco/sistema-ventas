# Sistema de ventas

## Instalar
**React Router**

`npm i react-router-dom`

**Font Awesome**

`npm i --save @fortawesome/fontawesome-svg-core`\
`npm install --save @fortawesome/free-solid-svg-icons`\
`npm install --save @fortawesome/react-fontawesome`

**Firebase**

`npm install firebase`

**jquery**

`npm install jquery`
